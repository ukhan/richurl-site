работайте с информацией в удобном Вам формате

Возможности расширения

Расширение Rich URL позволяет:

  • копировать информацию с
    • текущей вкладки
    • всех вкладок текущего окна
    • вкладок всех открытых окон браузера
  • отправлять информацию с текущей вкладки по электронной почте
  • публиковать информацию с текущей вкладки в
    • Facebook
    • Twitter
  • выполнять запросы к API сервисов для, например
    • превращения URL текущей страницы в QR код
    • поиска в DuckDuckGo, Wikipedia, YouTube, Amazon, …
    • создания копий на archive.org
    • …

В расширении можно создать любое количество действий, упомянутых выше, и вызывать их через меню или комбинации клавиш.

В каждом действии можно гибко настроить какая именно информация будет копироваться, отсылаться или публиковаться и в каком виде. Для использования доступны следующие переменные:

  • адрес страницы;
  • заглавие страницы;
  • выделенный на странице текст;
  • весь текст страницы.

Каждая переменная может быть отдельно настроена, например, для адреса страницы доступны:

  • преобразование unicode-символов в читаемый текст (когда вместо https://ru.wikipedia.org/wiki/%D0%A3%D0%BA%D1%80%D0%B0%D0%B8%D0%BD%D0%B0 получаем https://ru.wikipedia.org/wiki/Украина);
  • очистка маркетинговых и других служебных меток;
  • дополнения адреса своими параметрами, что может быть полезно, например, при подготовке линков для партнерских программ;
  • сокращение адреса с помощью одного из сервисов bit.ly, is.gd, clck.ru.
