Extension installation

To install the Rich URL extension, go to the Chrome Web Store via this link and click “Add to Chrome” ❶

Confirm the permissions1 by clicking on the “Add extension” button ❷

After that, the icon of the extension ❸ will appear in the browser panel and you can proceed to the configuration and use of the extension.

Find out more about:

  • Configuring the extension
  • How to use the extension
  1. Permission “Read and change all your data on the websites you visit” is requested by the extension to embed a script on the pages that provides keyboard shortcuts and access to the selected text on the page. “Display notifications” – to inform about the features of new versions of the extension. “Modify data you copy and paste” – to access the clipboard.
