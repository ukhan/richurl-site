work with information in a format that suits you best

The purpose of the extension

Rich URL is an extension for Google Chrome browser (and other browsers based on Chromium), which will help you to copy, share or save information from viewed pages in one or two clicks.

Why use the extension for copying if there is a standard shortcut of Ctrl+C/Cmd+C for this purpose, ask you?

With the extension, it will be a little faster and more comfortable.

Let’s suppose that it is necessary to copy from page a fragment of the text and to accompany its address of the page, that then not to forget where the text is taken from. Let’s calculate the number of operations that need to be performed for this purpose with and without the extension.

#Without extensionWith extension Rich URL
1.Select a text fragmentSelect a text fragment
2.Copy a text fragmentPress the extension button or keyboard shortcut
3.Paste a text fragmentPaste a text fragment together with the page address
4.Copy page address
5.Paste page address

When sending the same text with a link by email, the difference may be even more noticeable.

#Без расширенияС расширением Rich URL
1.Select a text fragmentSelect a text fragment
2.Copy a text fragmentPress the extension button or keyboard shortcut
3.Create a new email in the mail clientEnter the recipient’s address1 in an automatically generated2 letter
4.Paste a text fragmentSend a letter
5.Copy page address
6.Paste page address
7.Enter or copy/paste the subject of the letter
8.Enter the address of the recipient
9.Send a letter
