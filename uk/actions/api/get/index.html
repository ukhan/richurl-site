Виконання GET запитів до API

За допомогою GET запитів можна, наприклад:

  • перетворити URL поточної сторінки в QR код 1
  • здійснити пошук виділеного фрагмента в
    • DuckDuckGo 2
    • Wikipedia 3
    • YouTube 4
    • Amazon 5
    • …
  1. https://api.qrserver.com/v1/create-qr-code/?size=150x150&data={{ url | clear }}
    Докладніше про цей API. [повернутись]
  2. https://duckduckgo.com/?q={{ selected }} [повернутись]
  3. http://ru.wikipedia.org/w/index.php?title=Special:Search&search={{ selected }} [повернутись]
  4. http://www.youtube.com/results?search_query={{ selected }} [повернутись]
  5. https://www.amazon.com/s?k={{ selected }} [повернутись]
